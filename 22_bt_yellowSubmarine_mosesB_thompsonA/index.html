Explanation of the navbar

At the top is a navbar. This can be used to route the user to different links and can contain a search bar. If you scroll, the Navbar will stay at the top of your screen. This can be made using the following code:

<nav id="navbar-example2" class="navbar navbar-light bg-dark sticky-top">
  <a class="navbar-brand text-light" href="#">Navbar</a>
  <ul class="nav nav-pills">
    <li class="nav-item">
      <a class="nav-link" href="#link">Link display</a>
    </li>
  </ul>
</nav>

Notice that you can include links in nav-items which route the user elsewhere.
Also notice how the navbar stays at the top of the screen when you scroll. This is because of the "sticky-top" tag in the initial nav declaration.
